Getting Started with IoTLogIQ
Administrator Dashboard

The goal of this tutorial is to demonstrate the basic usage of the most popular IoTLogIQ’s Administrator Dashboard features. You will learn how to:

  • Create Customers, locations, areas, assets, devices
  • Assign devices to different assets.
  • Create Users.
  • Assign Users Role and Permissions
This Dashboard is only accesible to super-admin users
1-Select desired company, location, area, assets

1-On the top of the panel you’ll find access to the different entities: companies, locations, areas, and assets that are available in each of the different selections.

With each selection you make the respectful subcategories will load on the following column.

Within each panel you will be able to add, edit or delete a new item to the category.

2-At the bottom section of the dashboard you can view all the locations and assets available
2- Create a new device

You can start the device creation process by clicking the icon on the top right of the panel.

This will open the following Pop Up where you'll be able to select the name, type, and label of the device.

 

Add, edit and delete buttons are available in all entities: Customers, location, areas, assets and devices
3- Assign the device
To assign the device, click the icon located at the top right of the panel.
This will open the following pop-up where you'll be able to select the specifications of the machine to which the device will be assigned.
4-Create a new User
Create a new user by clicking on the plus icon in the top right corner.

Complete the user information, select their role, and assign their permissions.

 

Depending on the user role selected, the available permissions will differ.

 

-Viewers: cannot perform CRUD operations.

-Supervisors: can edit and delete most of the entities.

-Admins: have access to all actions.

This will create an activation link that can be sent to the account owner so that they can set their email and password to gain access to all the features available for the account.
5-Log In
As a super administrator, you will be able to log in as any user of your company to see their dashboard and the devices available to them.
